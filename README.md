# ClipStream 🎬

**ClipStream** is a simple video sharing and streaming backend project — inspired by YouTube.  
It allows users to upload, watch, and manage videos with features like likes, playlists, and subscriptions.

### 🔧 Features

- User login and signup
- Video upload and playback
- Like system
- Watch history and saved videos
- Clean, dark-themed UI support

### 🛠️ Tech Stack

- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Storage:** Cloudinary
- **Frontend (optional):** vue.js

### 🚀 Setup

1. Clone this repository
   ```bash
   git clone https://github.com/usama-sagheer/clipstream-backend
   ```
2. Install dependencies
   ```bash
   npm install
   ```
3. Start the server
   ```bash
   npm start
   ```
